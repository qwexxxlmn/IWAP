О языке JavaScript

JavaScript - это язык программирования, который используется для создания динамических веб-сайтов и приложений.

Зачем нужен JavaScript?

  • Делать сайт интерактивным
  • Создавать веб-приложения
  • Создавать игры
  • Создавать мобильные приложения
  • И многое, много другое

Когда JavaScript не пригодится?

  • В сложных вычислительных задачах
  • В низкоуровневом системном программировании
  • В приложениях, которые работают в режиме реального времени со строгими временными требованиями
